Qu'est-ce que HTML?

HTML est le langage de balisage standard pour la création de pages Web. HTML signifie Hyper Text Markup
HTML décrit la structure des pages Web en utilisant le balisage
Les éléments HTML sont les blocs de construction de pages HTML
Les éléments HTML sont représentés par des balises
balises HTML morceaux d'étiquette de contenu tels que « rubrique », « paragraphe », « table », et ainsi de suite Les navigateurs ne présentent pas les balises HTML, mais les utiliser pour rendre le contenu de la page

Documents HTML


Tous les documents HTML doivent commencer par une déclaration de type de document: '< !DOCTYPE html> .
Le document HTML lui - même commence par < html> et se termine par .
La partie visible du document HTML est entre < body> et .

La balise < a> définit un hyperlien

,
qui est utilisé pour relier d'une page à l'autre.
L'attribut le plus important de l'élément
< a> est l'attribut href, qui indique la destination du lien.
Par défaut, les liens apparaissent comme suit dans tous les navigateurs:
Un lien est unvisited bleu et souligné
Un lien est visité et violet souligné
Un lien actif est souligné et rouge
Le < area> définit une zone à l'intérieur d'une image-map (e-plan d'image est une image avec des zones cliquables).

L'élément < zone> est toujours niché dans une balise < carte>

.
Remarque: L'attribut usemap dans le < img> tag est associé à la < map> Nom élément attribut, et crée une relation entre l'image et la carte.

La balise < audio>


définit son, comme la musique ou d'autres flux audio. À l'heure actuelle, il existe 3 formats de fichiers pris en charge pour l'élément < audio>: MP3, Wav, Ogg

La balise < code>


est une balise de phrase. Il définit un morceau de code informatique. Astuce: Cette balise est pas dépréciée,
mais il est possible d'obtenir un effet plus riche avec CSS

La balise < div>


définit une division ou une section dans un document HTML.
La balise < div> est utilisé pour les éléments de bloc-groupe pour les mettre en forme avec CSS.

La balise < input>

spécifie un champ d'entrée où l'utilisateur peut entrer des données.
< Input> éléments sont utilisés dans un élément < form>
pour déclarer commandes d'entrée qui permettent aux utilisateurs d'entrer des données.
Un champ d'entrée peut varier de plusieurs façons, en fonction de l'attribut type.
